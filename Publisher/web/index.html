Sobre o Aplicativo

O aplicativo Meu carro, conhecido anteriormente como Flex Profile foi atualizado para resolver alguns problemas referente ao modo antigo de seleção de números. Esse aplicativo te ajudará a decidir qual o combustível mais vantajoso na hora do abastecimento. Esse calculo é feito pelo custo do KM/L. Para o mesmo funcionar, os abastecimento deverão ser de "completar o tanque até enche-lo", ele necessita disso para o calculo correto do consumo.
Caso tenha migrado do Flex Profile, o modo de funcionamento é o mesmo, porém com melhorias na inserção dos dados e também o novo menu "Óleo", onde o usuário poderá cadastrar as trocas de óleo dos veículos.
Para o usuário novo, é necessário inserir um veiculo para que o calculo seja feito. Isso pode ser feito adicionando um novo veículo ao aplicativo, puxando o menu lateral para a direita e indo até a opção Veículos.
Dentro dessa opção o usuário poderá ver uma lista com os veículos cadastrados, e selecionando o botão de "+" poderá adicionar um veículo. Em adicionar veículo, informe o nome do carro, os combustíveis e seus consumos e a capacidade do tanque do mesmo em litros (caso tenha GNV, adicione também a metragem cúbica do cilindro).
Com o carro cadastrado, informe o valor do litro de cada combustível que o calculo é feito automaticamente ao modificar o preço dos mesmos.

Limitações

Para que o app calcule corretamente se faz necessário que o abastecimento de combustivel seja sempre de "encher o tanque", para que o calculo seja feito com os litros inseridos pra encher o tanque.

Devido ao modo de funcionamento ou até, por limitações fisicas algumas funções possuem um funcionamento diferente do esperado: Exibição de Média de consumo quando há troca de GNV e outros combustiveis e vice-versa: Quando essa troca é feita, há uma quebra no fluxo de funcionamento da medição de consumo,: Basicamente a conta feita para medir o consumo é :

 (KM do Ultimo Abastecimento - KM do Atual abastecimento / Litros do atual abastecimento )

Quando a troca de combustiveis é feita, não há como o aplicativo ou o usuário saber quantos litros foram inseridos para rodar o trecho (KM - KM atual), para isso o app irá assumir que o combustivel anterior foi usado até o final, com isso os litros utilizados na formula será do Ultimo Abastecimento e não do Abastecimento Atual, porém o KM será do Abastecimento atual. Quando isso ocorrer, consumo será exibido em vermelho, com um * na frente do valor informado. Caso você tenha uma outra maneira de medição para essa limitação e quiser comparilhar com o App, entre em contato explicando o seu ponto de vista , para que ele possa ser implementado no App. 

Calculo de consumo Normal

O cálculo é feito utilizando os dois abastecimentos, usando o KM atual, KM anterior e a litragem atual de combustivel , para se obter a média de Consumo do abastecimento Anterior.

Calculo de consumo Estimado                   

O cálculo é feito utilizando os dois abastecimentos, usando o KM atual, KM anterior e a litragem anterior de combustivel , para se obter a média de Consumo do abastecimento Anterior.
Note que quando esse calculo estimado é realizado, o Consumo e distancia percorrida ficam marcados em vermelho com um asterisco, indicando que o método de calculo foi estimado.
